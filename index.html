Cara buat: Dashboard Cloudinary → Settings → Upload → Add upload preset → set Signing Mode ke Unsigned → Save.
Info
Free tier Cloudinary memberi 25 credits/bulan (1 credit = 1 GB storage atau 1 GB bandwidth). Gambar komisi biasa (JPG/PNG) biasanya 1–5 MB — lebih dari cukup untuk pemakaian pribadi.

Format didukung: JPG, PNG, WEBP, GIF, BMP, TIFF, PSD, SVG, PDF, dan lainnya.
